# UX Principles

## Core Rule

Every screen must answer:

"What should I do next?"

---

## Design Rules

- No page should require explanation
- Every page must have:
  - clear purpose
  - clear next action
- Avoid unnecessary clicks

---

## Navigation

- Keep navigation predictable
- Avoid deep nesting
- Max 3 levels

---

## Decision UX

This is NOT a dashboard tool.

This is a DECISION ENGINE.

Every page must guide:
- what is wrong
- why it matters
- what to fix

---

## Error Handling

- Always show user-friendly messages
- Never expose system errors
