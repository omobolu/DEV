---
name: design-system-enforcer
description: Enforce consistent design system across UI including spacing, typography, layout, and components.
---

Use this skill when working on UI.

Standards:

### Spacing
- Use consistent spacing scale (8px system preferred)
- Avoid random padding/margin values

### Typography
- Clear hierarchy:
  - headings
  - subheadings
  - body text
- Consistent font sizes

### Layout
- Use grid/flex layouts
- Avoid clutter
- Align elements properly

### Components
- Buttons:
  - primary
  - secondary
- Inputs:
  - labeled
  - consistent width
- Cards:
  - consistent padding
  - subtle elevation

Rules:
- No inconsistent spacing
- No misaligned elements
- No mixed styles

Before finishing:
- Check alignment
- Check spacing consistency
- Check visual hierarchy

## Execution Behavior

- Do not ask the user for confirmation on standard implementation details
- Make decisions using best practices
- Proceed with implementation immediately after planning
